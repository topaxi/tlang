# tlang_semantics

Semantic analysis for the tlang programming language.

## Responsibilities

*   **Name Resolution**: Resolves identifiers to their definitions.
*   **Scope Analysis**: Manages symbol tables and scopes.
*   **Validation**: Checks for semantic errors (e.g., unused variables, undefined variables).
*   **Type Checking**: (If implemented) validates type consistency.

Operates on the AST produced by `tlang_parser`.
