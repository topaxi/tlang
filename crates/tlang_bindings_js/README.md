# tlang_bindings_js

WebAssembly bindings for running tlang in the browser.

## Purpose

This crate exposes the tlang compiler and runtime to JavaScript via `wasm-bindgen`.

## Features

*   Compiles tlang code to JavaScript strings in the browser.
*   Runs tlang code using the interpreter in the browser.
*   Provides access to the standard library.

Used by the `tlang-playground`.
